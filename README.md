## 🚀 Описание
**eCommerce-Application** — приложение для розничной онлайн-продажи меда и продуктов пчеловодства, простое и понятное для пользователя. Приложение позволяет просмотреть каталог продуктов, выбрать понравившийся продукт и заказать его доставку на указанный адрес с оплатой онлайн. Доставка осуществляется по всему миру, возможна оплата в различных валютах.
Реализованы:
- форма регистрации и входа
- поиск по сайту 
- добавление товаров в корзину
- оформление заказа и форма оплаты
- рейтинг товаров в зависимости от оценки пользователей (1-5 звезд)
- рейтинг покупателей и система скидок
- возможность добавлять отзывы к товарам
- форма обратной связи
Приложение разработано с применением современныхпрактик для максимальной доступности. Для защиты данных используются сертификаты SSL.
## 🏠 Структура приложения
1. **Главная страница**
На главной странице приложения расположена основная информация о магазине. Вверху страницы находятся панель навигации, форма входа и регистрации. Есть возможность просматривать товары и перейти на страницу каталога, кнопка CTA, перенаправляющая в корзину к форме заказа выбранных товаров. Здесь также размещается информация о текущих акциях и скидках. В нижнем правом углу страницы находится всплывающее окно чат-бота для помощи и связи с оператором.
    - **Страница входа и регистрации**
    На странице входа находятся поля для ввода логина и пароля со встроенной валидацией и кнопка отправки данных. При вводе корректной информации пользователь перенаправляется на главную страницу, в правом верхнем углу появляется его логин.
    - **Профиль пользователя**
    В профиле пользователя находится личная информация: ФИО, пол, дата рождения, список адресов, платежная информация. Здесь также сохраняется информация о времени, прошедешем с момента регистрации, рейтинге покупателя, предыдущих заказах и доставках, участии в программе лояльности, доступных личных скидках и специальных предложениях. Пользователь может выбрать в настройках сайта светлую или темную тему, язык, версию для слабовидящих.
2. **Каталог товаров**
В каталоге товаров представлен список товаров, представленных в виде интерактивных карточек, разделенных по категориям. Вверху страницы есть фильтр для отображения карточек, где можно выбрать одну или несколько категорий, сортировку по популярности, стоимости (сначала дорогие или сначала дешевые), новизне.
    - **Карточка товара**
    Интерактивные карточки товаров представлены в каталоге в сокращенном виде, включающем название товара, фото, рейтинг, цену и кнопку добавления в корзину, участие в акциях и специальных предложениях. По клику открывается полная версия, включающая также подробную информацию о товаре - характеристики, описание, доступные варианты и отзывы, если они есть. Если товар был заказан и доставлен пользователю, после подтверждения доставки открывается возможность оценки товара (1 - 5 звезд) и поле для написания и отправки отзыва.
    - **Корзина**
    В корзине находятся все карточки товаров, выбранные и добавленные пользователем. Здесь можно просматривать выбранные товары, изменять их количество или удалять из корзины. Пользователю видна общая цена выбранных товаров и размер личной скидки. С помощью калькулятора можно рассчитать стоимость доставки. Из корзины можно перейти к форме заказа.
        - калькулятор стоимости доставки
        Состоит из поля ввода с выпадающего списком с названиями населенных пунктов, поля ввода веса товара (подсчитывается автоматически, как и общая стоимость), поля выбора способа доставки и вывода результата.
        - форма заказа
        Форма заказа включает в себя список выбранных товаров, выбор способа оплаты и доставки. Адреса и платежная информация экспортируются из профиля пользователя. Нажав на кнопку подтверждения, пользователь может перейти к форме оплаты.
        - форма оплаты
        Форма оплаты включает поле для ввода кода подтверждения оплаты, который приходит на мобильный телефон пользователя, и кнопку подтверждения.
3. **"О нас"**
Здесь размещается информация о компании - владельце магазина: название, стаж работы, миссия и ценности, история создания, фотографии, успешные кейсы, подборка отзывов покупателей. 
Внизу страницы размещается информация о разработчиках: имена и роль каждого в создании проекта, ссылки на страницы в Github, год разработки. Здесь же размещен логотип RS SCHOOL со ссылкой на школу.
4. **Контакты**
Здесь размещаются адреса и телефоны головного офиса компании, всех магазинов и филиалов, ссылки на социальные сети и мессенджеры. Здесь также находится форма обратной связи, через которую можно отправить сообщение на e-mail магазина.
    - форма обратной связи
5. **Новости**
В этом разделе можно разместить мини-блог, где будут публиковаться новости компании и SEO-оптимизированные тематические статьи для продвижения в рейтинге поиска, привлечения и удержания целевой аудитории.
    -**Страница записи**
    Каждая новость или запись в блоге представляет собой структурированную статью, в которой могут размещаться ссылки или кнопка CTA с предложением узнать больше об акциях или продуктах, при переходе по ссылке или клику по кнопке открывается карточка товара или ссылка на страницу акции, на которой можно увидеть список товаров, участвующих в акции.

## 💻 Локальный запуск проекта
1. Установите [Node.js](https://nodejs.org/en/download)
2. Клонируйте этот репозиторий: https://github.com/dongl1n/eCommerce.git
3. Перейдите в папку склонированного репозитория с помощью `cd ./eCommerce`
4. Используйте `git branch -r`, чтобы получить и просмотреть удаленные ветки репозитория
5. Определитесь в какой ветке будете работать:

- Выберите соответствующую ветку для добавления планируемого функционала с помощью `git checkout название ветки`
- Или перейдите в ветку develop, а затем создайте новую ветку:
  ```
  git checkout develop
  git checkout -b название ветки
  ```
6. Перейдите в папку проекта с помощью `cd ./eCommerce`, обратите внимание, что текущий путь выглядит так примерно так `C:\Users\UserName\...\eCommerce\eCommerce`
7. Для установки всех зависимостей используйте `npm install`
8. Используйте [скрипты](#scripts) для работы с проектом
9. Сопровождайте разработку полными и понятными коммитами согласно [гайдлайну](https://docs.rs.school/#/git-convention)
