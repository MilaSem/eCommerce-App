## 🚀<a id="description"></a> Описание

**eCommerce-Application** — приложение для розничной онлайн-продажи мёда и продуктов пчеловодства, простое и понятное для пользователя. Приложение позволяет просмотреть каталог продуктов, выбрать товар, заказать и оплатить. Доставка в любую точку мира, оплата в различных валютах.

**Основные характеристики:**

- 💰 работа на базе CommerceTools
- 📱 адаптивность
- 🔎 доступный интерфейс

**Реализованы:**

- регистрация пользователя и вход в систему
- поиск по сайту
- категории товаров и сортировка
- добавление товаров в корзину
- оформление заказа и форма оплаты

**Дополнительные функции:**

- рейтинг товаров
- программа лояльности
- калькулятор стоимости доставки
- возможность добавлять отзывы
- чат с онлайн-помощником

## 🐝<a id="structure"></a> Структура приложения

1.  📝 **Страница входа и регистрации**  
    На странице входа находятся поля для ввода логина и пароля со встроенной валидацией и кнопка отправки данных. При вводе корректной информации пользователь перенаправляется на главную страницу.

2.  🏠 **Главная страница**  
    На главной странице приложения расположена основная информация о магазине. Здесь можно просмотреть товары и перейти на страницу каталога, ознакомиться с текущими акциями и скидками. Вверху страницы находится панель навигации. Есть кнопка CTA, перенаправляющая в корзину к форме заказа.  
    _Опционально: в нижнем правом углу страницы можно разместить всплывающее окно чат-бота_

3.  📕 **Каталог товаров**  
    В каталоге представлен список товаров в виде интерактивных карточек, разделённых по категориям. Вверху страницы есть фильтр, где можно выбрать одну или несколько категорий, сортировку по популярности, стоимости (сначала дорогие или сначала дешевые), наличию скидки.

4.  🎁 **Подробная страница товара**  
    Интерактивные карточки товаров представлены в каталоге в сокращённом виде, включающем название товара, фото, цену и кнопку добавления в корзину. Опционально: рейтинг, скидки и специальныe предложения. По клику открывается полная версия, включающая подробную информацию о товаре: характеристики, описание, доступные варианты и отзывы.  
    _Опционально: если товар был заказан и доставлен пользователю, после подтверждения доставки открывается возможность оценки товара (1 - 5 звезд) и написания отзыва._

5.  👤 **Профиль пользователя**  
    В профиле пользователя находится личная информация: ФИО, пол, дата рождения, список адресов, платёжная информация. Пользователь может выбрать в настройках сайта светлую или темную тему, язык, версию для слабовидящих.  
    _Опционально: сохраняется информация о времени регистрации, рейтинге покупателя, предыдущих заказах, участии в программе лояльности, доступных личных скидках и специальных предложениях._

6.  🛒**Страница корзины**  
    В корзине находятся все карточки товаров, выбранные и добавленные пользователем. Здесь можно просматривать выбранные товары, изменять их количество или удалять из корзины. Пользователю видна общая цена выбранных товаров и размер личной скидки. Из корзины можно перейти к форме заказа.

- 📋 Форма заказа  
  включает в себя список выбранных товаров, выбор способа оплаты и доставки. Адреса и платёжная информация экспортируются из профиля пользователя. Нажав на кнопку подтверждения, пользователь переходит к форме оплаты.
- 💳 Форма оплаты  
  включает поле для ввода индивидуального кода и кнопку подтверждения оплаты.
- ✈️ _Опционально: калькулятор стоимости доставки_
  _Состоит из поля ввода с выпадающим списком с названиями населённых пунктов, поля ввода веса товара (подсчитывается автоматически, как и общая стоимость), поля выбора способа доставки и вывода результата._

7. 👦👧 **Страница "О нас"**  
   Здесь собраны все сведения о компании: название, история, миссия и ценности, фото, отзывы, контактная информация: адреса, телефоны, почта, ссылки на социальные сети и мессенджеры.
   Внизу страницы размещается информация о разработчиках: имена и роль каждого в создании проекта, ссылки на страницы в Github, год разработки. Здесь же размещен логотип RS SCHOOL со ссылкой на школу.

## 📚 <a id="technologies"></a> Стек технологий

1. **Настройка Бэкенд:**

- CommerceTools API

   Проект — SPA, написанный на React.  
   Развертывается на Netlify.  
   Для разработки используются:

- React-router
- Swr
- TypeScript
- Vite
- ESlint
- Prettier
- Husky
- Commitlint
- Lint-staged
- Пакеты для тестов
- CSS-модули (поддерживаются Vite)  
  Для работы с платформой CommerceTools:
- @commercetools/platform-sdk
- @commercetools/ts-client

3. **Тестирование**

- Vitest (аналог Jest, оптимизированный для Vite)

4. **Библиотеки**

- Zustand (библиотека для управления состоянием)
- Ant Design (библиотека компонентов UI для React)

5. **Управление**

- Git / Github

## 🚩 <a id="scripts"></a>Скрипты

- `"dev"`: `"vite"` — запускает dev-сервер Vite;
- `"build"`: `"tsc -b && vite build"` — собирает продакшн-версию проекта: компилирует Typescript, затем создает bundles и запускает проект с помощью Vite;
- `"lint"`: `"eslint"` — проверяет Javascript/Typescript код на ошибки и стилевые нарушения;
- `"format"`: `"prettier --write --ignore-unknown ."` — форматирует код проекта, согласно настройкам;
- `"stylelint"`: `"stylelint '**/*.css' --fix"` — проверяет и автоматически исправляет стили CSS;
- `"test"`: `"vitest"` — запускает тесты через Vitest;
- `"preview"`: `"vite preview"` — запускает локальный сервер для предпросмотра продакшн-версии приложения;
- `"prepare"`: `"husky"` — устанавливает Git-хуки с помощью husky для автоматического запуска lint, test и т.д. перед коммитами.

## 💻 <a id="start"></a> Локальный запуск проекта

1. Установите [Node.js v.22(LTS)](https://nodejs.org/en/download)
2. Клонируйте этот репозиторий: https://github.com/MilaSem/eCommerce-App.git
3. Перейдите в папку склонированного репозитория:`cd eCommerce-App`
4. Установите зависимости: `npm install`
5. Используйте `git branch -r`, чтобы получить и просмотреть удаленные ветки репозитория
6. Выберите ветку:

- для работы с существующей веткой: `git checkout название_ветки`
- для создания новой ветки из develop:
  `git checkout develop`
  `git checkout -b feature/your-feature`

7. Запустите проект: `npm run dev`
8. Используйте [скрипты](#scripts) для работы с проектом
9. Сопровождайте разработку полными и понятными коммитами согласно [гайдлайну](https://docs.rs.school/#/git-convention)
